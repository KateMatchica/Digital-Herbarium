# The Digital Herbarium
A Virtual Reality educational archive preserving extinct flora.

## Project Description
This application creates a virtual learning environment where users can explore 3D holographic representations of extinct plant life. It utilizes A-Frame to render an immersive sci-fi laboratory.

## Tech Stack
- A-Frame (VR Framework)
- HTML5 / JavaScript
- GLTF 3D Models

## Installation
1. Clone the repository.
2. Run `npm install`
3. Run `npm start`
4. Open `localhost:3000` in your browser.
